Jawaban:

C. Descriptive text

Note :

Descriptive text lebih tepat yang memiliki tujuan untuk mendeskripsikan orang/tempat/benda secara spesifik.

Narrative text has a purpose to amuse/entertain the readers

Recount text has a purpose to retell the past event or experience

Procedure text has a purpose to explain the steps how to use or how to make something in detail.

Penjelasan:

DESCRIPTIVE TEXT

  • Genre teks yang menjabarkan suatu benda/orang/tempat dan hal lainnya secara spesifik/khusus.

Social function/ communicative purpose :

  • to describe people/place/things specifically ( untuk mendeskripsikan orang/tempat/hal -hal secara khusus )

Generic Structure  

1.  Identification  

  • Bagian paragraf ini  terletak di awal sebagai pengenalan siapa/apa yang akan dijabarkan dalam teks

2.  Description  

  • Bagian paragraf selanjutnya menjabarkan ciri khas-nya bagaimana dari orang/benda/tempat/hal yang di bicarakan.

3. Conclusion  

  • Paragraf akhir berisi kesimpulan , ungkapan  dari si penulis. Bagian ini optional ( Bisa dibahas atau pun tidak )

LANGUAGE FEATURE ( Unsur kebahasaan )

1. Penggunann simple present tense

  • Dikarenakan penjabaran berdasarkan fakta kebenaran, maka tenses yag digunakan adalah simple present tense baik dalam bentuk kalimat verbal dan nomina.

2.  Penggunaan adjective ( kata sifat )

  • Penjabaran karakter/ciri khas lebih dominan dalam teks deskripsi

3. Penggunaan linking verb ( relating verb )

  • Ini merupakan kata kerja pengikat subyek. Biasanya berupa BE present ( is, am,are, will, can , and etc ) sebagai pengikat kata sifat dan keterangan dari subyek yang dijabarkan.
