Jawaban: #1:

Dalam permainan bulu tangkis ada banyak sekali macam kata yang dipergunakan, misansalnya seperti : rubber set serta straight set. Selain itu masih ada berbagai istilah lainnya.

  • Straight Set : Pertandingan yang terselesaikan pada 2 set pribadi.
  • Rubber Set : Keadaan waktu dua set sebelumnya seimbang (saling mengalahkan), maka dilakukan set ke 3 menjadi penentuan set.

Penjelasan:

ISTILAH SERVIS

  • Server dan Receiver: dua istilah ini digunakan buat menyebut atlet yg melakukan serivis (server) serta penerima servis (receiver).
  • Long, Short, serta Wide: dalam pertandingan buulutangkis, kita lebih mengenal dengan istilah long, short, serta wide untuk menyatakan servis yg gagal. Shuttlecock yang jatuh di depan service box lawan dinamakan short. umumnya wasit akan berkata “Service too short.” jika shuttlecock jatuh memanjang ke luar lapangan, maka dinamakan long, sementara bila melebar ke luar dinamakan wide.
  • Service Over: digunakan ketika server kalah dalam sebuah rally. sehingga giliran servis harus berpindah ke versus.
  • Let: Let diberikan ketika penerima servis belum siap sementara servis sudah dilakukan. jika dinyatakan ‘let’ pemain tidak menambah score-nya serta servis harus diulang.

ISTILAH SCORE

  • Love: pada pertandingan bulutangkis, score 0 dinyatakan menjadi ’love’.
  • All: Jika ke 2 (pasang) pemain mempunyai score yg sama maka dinyatakan menjadi ‘all’. contohnya waktu kedudukan score dua-2, maka dinamakan ‘two all’.
  • Deuce: Bila kedudukan score 20-20 maka permainan dinyatakan menjadi ‘deuce’. merupakan, galat satu (pasang) pemain harus bisa memimpin dua poin lebih buat memenangkan pertandingan.

ISTILAH PERMAINAN

  • Rally: sebuah rally pertandingan dimulai saat keliru satu pemain melakukan servis. Rally berakhir Bila bola menyentuh tanah atau Bila pemain dinyatakan fault. Pemenang rally akan diberi satu score.
  • Game/ Set: pada pertandingan bulutangkis, kita harus mendapatkan score 21 buat memenangkan pertandingan. Bulutangkis mempunyai 2-tiga sesi pertandingan, satu sesi pertandingannya dinamakan sebagai set atau game.
  • Match: pertandingan bulutangkis memiliki 2-tiga game/set. Pemain dinyatakan memenangkan pertandingan Jika dia berhasil menang 2 set. tapi Jika ke 2 pemain masing-masing memenangkan satu game, maka wajib diadakan game lanjutan yg dinamakan match menjadi penentu pemain mana yg akan keluar menjadi pemenang.
  • Straight Set : Pertandingan yang terselesaikan pada 2 set pribadi.
  • Rubber Set : Keadaan waktu dua set sebelumnya seimbang (saling mengalahkan), maka dilakukan set ke 3 menjadi penentuan set.
  • Stroke/ pukulan: saat pemain mengayunkan raket, maka dianggap menjadi ‘pukulan’ atau ‘stroke’.
  • Clear Lob: memukul shuttlecock setinggi ke arah versus dinamakan menjadi ‘lob'.
  • Drop Shot: pukulan yg enggak terlalu keras serta menunjuk ke lapangan lawan.
  • Smash: pukulan yg sifatnya kencang dan ofensif ke arah lawan. umumnya arah pukulan smash menukik dengan cepat.
  • Drive: pukulan cepat tapi mendatar ke arah versus.
  • Net Shot: pukulan ini dilakukan dekat dengan net sebagai akibatnya diklaim sebagai net shot.

ISTILAH OFFICIAL

  • Wasit Turnamen/ Tournament Referee: artinya seorang yang ditugaskan buat menentukan peraturan pada pertandingan. Bila pemain tidak sepakat dengan keputusan umpire, mereka bisa memanggil wasit turnamen.
  • Match Umpire: tugas umpire buat memimpin serta memastikan pertandingan berjalan dengan lancar.
  • Service Judge: duduk berseberangan menggunakan umpire dan tugasnya buat melihat servis yg dilakukan oleh pemain. Service judge berhak memutus service fault Bila pemain melakukan kesalahan ketika servis.
  • Hakim Garis/ Line Judge: duduk di sekeliling lapangan dan bertugas buat melihat apakah bola jatuh pada luar atau di dalam lapangan.
